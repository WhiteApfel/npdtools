НПДТулзы приветствуют тебя!¶

Инструмент для работы с неофициальным и кривым API от ФНС для самозанятых.

Установка¶

Через pip

python -m pip install -U npdtools

Из исходников:

git clone https://gitlab.com/whiteapfel/npdtools.git
cd npdtools
python setup.py install
